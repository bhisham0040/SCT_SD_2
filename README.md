🎯 Mind Reader — Number Guessing Game

A Python number guessing game with an impressive GUI built using tkinter.
Made as Task 2 of the SkillCraft Technology internship.

Features


4 difficulty levels (Easy → Legend)
Temperature hints (🧊 ice cold to 🔥 burning hot)
Animated particle background
Score & streak system
Attempt tracker with visual dots


Run
bashpython number_guessing_game.py

Requires Python 3.7+ with tkinter
